A lightweight bot verification page used with the bot-check GitHub Action. Simply set it as the verification endpoint, and users will be prompted to complete a Cloudflare Turnstile check before being redirected to the destination.

> Built for [Bot Check Action](https://github.com/dohyeon5626/bot-check-action)  
> Works with [Serverless Functions](https://github.com/dohyeon5626/serverless-functions/tree/main/bot-check)
